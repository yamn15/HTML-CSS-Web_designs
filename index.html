pタグについて

"pタグ"は、Paragraph(段落を示す)

"pタグ"が連続すると、自動的に改行される

写真の挿入について

パスとファイル名を指定して、写真を表示させる

パスには相対パスや絶対パスなどの指定方法がある

画像の大きさの調整は要復習

リンクについて

リンクを生成する際のタグがある

yahoo

画像にリンクを張ってみる

リンクタグで、画像を挟めばいい

リストの表示について

箇条書きリストを作成する際は、ul(Unordered List)タグとliタグを利用する

  • 卵(3個)
  • 醤油(大さじ1)
  • みりん(大さじ1)
  • 塩(少々)

番号のついたリストは、ol(Ordered List)タグとliタグを利用する

  1. 卵(3個)
  2. 醤油(大さじ1)
  3. みりん(大さじ1)
  4. 塩(少々)

フォームを作成する

フォームは複数の機能を組み合わせる

フォーム欄を作るformタグ

以下が主な属性

属性 用途
action データ送信先ページを指定
method データの転送方法の指定。主にgetかpostを入力
name フォームの名前を指定

example

フォームで使うパーツ

inputやtypeがある

一行テキスト入力欄

inputタグを使用する

入力欄にあらかじめテキストを入力しておくこともできる

以下が主な一行テキスト入力欄の属性

属性値 用途
text 1行のテキスト
search 検索するときのテキスト
email メールアドレス
tel 電話番号
url WebサイトのURL

ラベルを使用する最中の注意点

識別名が必ず一対にならなければならない。重複できない。半角英数字のみ使える。数字や記号から始まる識別名は使えない

フォームのラベルを作るラベルタグ

以下がラベルタグ

ブロック要素でグループ分け

Webページの中身はいくつか要素に分けられる

ページ上部のヘッダー部分 headerタグ

ロゴ画像やページタイトルなどが配置される。head要素とは異なるので要注意

ナビゲーションメニューを作るnavタグ

メインのナビゲーションメニューを囲む。

headerタグの中に含まれることが多い。基本的にメインではないメニューには使用しない。

読み物、記事の部分を作るarticleタグ

テーマを持ったグループを作るsectionタグ

ページのメインコンテンツ部分を作るmainタグ

メインコンテンツではない補足情報asideタグ

ページ下部のフッター部分を作るfooterタグ

意味を持たないブロック要素を作るdivタグ
