HTMLファイルの作成方法

見出しについて

2番目の見出しを表示

見出しの使い方

一つのページには、h1タグは1つにするといい

一つのページには、h4くらいまでを使用するのが一般的

pタグについて

"pタグ"は、Paragraph(段落を示す)

"pタグ"が連続すると、自動的に改行される

写真の挿入について

パスとファイル名を指定して、写真を表示させる

パスには相対パスや絶対パスなどの指定方法がある

画像の大きさの調整は要復習

リンクについて

リンクを生成する際のタグがある

yahoo

画像にリンクを張ってみる

リンクタグで、画像を挟めばいい

リストの表示について

箇条書きリストを作成する際は、ul(Unordered List)タグとliタグを利用する

  • 卵(3個)
  • 醤油(大さじ1)
  • みりん(大さじ1)
  • 塩(少々)

番号のついたリストは、ol(Ordered List)タグとliタグを利用する

  1. 卵(3個)
  2. 醤油(大さじ1)
  3. みりん(大さじ1)
  4. 塩(少々)

表を作成する

表組は複数のタグを合わせて使用する

プラン名 料金

にこにこプラン 月額1,980円

縦にセルをつなげる

rowspan属性を追加する

フォームを作成する

フォームは複数の機能を組み合わせる

フォーム欄を作るformタグ

以下が主な属性

属性 用途
action データ送信先ページを指定
method データの転送方法の指定。主にgetかpostを入力
name フォームの名前を指定

example

フォームで使うパーツ

inputやtypeがある

一行テキスト入力欄

inputタグを使用する

入力欄にあらかじめテキストを入力しておくこともできる

以下が主な一行テキスト入力欄の属性

属性値 用途
text 1行のテキスト
search 検索するときのテキスト
email メールアドレス
tel 電話番号
url WebサイトのURL

ラジオボタンを作る

以下が主な属性

属性 用途
name ラジオボタンの名前
value 送信される選択肢の値
checked 最初から選択されている状態にするときに指定

ラジオボタンの例

male female others

チェックボックスを作る

以下が主な属性

属性 用途
name チェックボックスの名前
value 送信される選択肢の値
checked 最初から選択されている状態にするときに指定

チェックボックスの例

red blue yellow green

送信ボタンを作る

以下が主な属性

属性 用途
name ボタンの名前
value ボタンに表示するテキスト

送信ボタンの例

送信ボタンに画像を使う

以下が主な属性

属性 用途
name ボタンの名前
source ボタンに使用したい画像のファイルパス、ファイル名
alt 画像を説明するテキスト
